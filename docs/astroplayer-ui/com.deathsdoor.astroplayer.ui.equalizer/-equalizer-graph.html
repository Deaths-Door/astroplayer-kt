EqualizerGraph

@Composable
fun EqualizerGraph(state: EqualizerGraphState, slider: @Composable (bandLevel: Int, frequency: Float) -> Unit = { bandLevel, _ -> EqualizerGraphSlider( state = state, bandLevel = bandLevel, ) }, modifier: Modifier = Modifier, enabled: Boolean = true, colors: SliderColors = SliderDefaults.colors())

A composable that represents the entire equalizer graph with multiple sliders.

Parameters

state

An EqualizerGraphState object that holds the data for the equalizer.

slider

A composable function that takes two arguments: * bandLevel: The index of the equalizer band (key in the frequencies map of the state). * frequency: The current frequency value (dB) for the band. This function is expected to render a single slider for the specified band. It's recommended to use the EqualizerGraphSlider function for this purpose.

modifier

Additional modifier to apply to the entire equalizer graph.

enabled

Whether the equalizer is enabled or disabled. This affects the color of the sliders.

colors

The color scheme to use for the sliders.
